# workpanel

## aim

核心目的是，说我现在能看到一个特定产品的bug\issue面板，我作为开发者需要修改项目，但是我希望coding agent解决这个问题是能够按照ADR规范，我可以控制，每一步决策都清晰的方法解决

## tools used

claude code \ codex \ cursor \ cursor cli \ pi coding agent \ opencode

## 目前的设想

设计一个可以控制的agent-human交互方法，主要核心是当前的工具效率太低，无法做到一次性解决多个并发issue并合并的问题。

就像是一个work编排的顶层agent，调用多个子agent使用类似于superpower方法，但是review这些我希望是自动化实现等等内容
